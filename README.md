# onちゃんの間違い探しゲーム（HTB秋の大感謝祭展示）

## 概要
このゲームは、似ている絵の違いを探して見つけた間違いの数を競う間違い探しゲームです。  
一部の問題には、絵の一部分が出たり消えたりする仕掛けがあり、最後にはランキングに反映されます。  
HTB秋の大感謝祭では、３日間で約1000人の来場者に遊んでもらいました。

## 遊び方
1. 画面に表示される絵を見て、違う箇所をクリックします。  
2. 一部の問題では、絵の一部が消えたり出たりする仕掛けもあり、見逃しやすい箇所に注意してください。  
3. 問題が難しい場合はスキップも可能です。  
4. 違いを見つけた数が記録され、最終的にはランキングに反映されます。


### プログラム面
- **タイトル画面でのプレイヤー名管理**  
  - TMP_InputField で名前入力を受け付け  
  - 未入力時は「名無し1,2…」の自動生成  
  - 入力文字数を6文字に制限し、PlayerPrefs に保存  

- **オフラインランキングの実装**  
  - ゲーム終了後にスコアを保存し、過去スコアと比較して高い方を保持  
  - 上位10件を表示、プレイヤー自身の順位も表示  
  - PlayerPrefs を使った軽量なデータ管理でサーバ不要  

- **ゲーム体験と連動した工夫**  
  - スコア更新がリアルタイムでUIに反映される  
  - ランキングにより、達成感と競争要素を強化  

### デザイン・演出面
- 家族連れやお子様がターゲットのイベントでの展示だったため、漢字を使わず文章を構成しました  
- onちゃんのブランドイメージを損なわないよう、セリフは公式に忠実に再現しました  
- 全体的なデザインは、お子様の目を引くかわいらしいデザインにしました
